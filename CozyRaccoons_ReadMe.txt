UKEMOCHI BY COZY RACCOONS
Ukemochi is a adventure game featuring a Soul Harvest mechanic combined with Hack-and-Slash combat.

USER CONTROLS
WASD - move player
R/T - rotate the player
F/G - scale the player
U - toggle debug draw
P - decrease volume
O - increase volume
M - mute volume
N - unmute volume
L - object clone

EDITOR CONTROLS
Q - Zoom in
E - Zoom out
arrow keys - Camera movement
